# End-to-End database development #

Have you wondered how the database schemas comes about? How do those database tables perfectly alinged with diferent keys -- that everyone does sql querying and joins etc come from? I got these answers from my DBMS class and it lies in Database design. I was always fascinated about how a database is built from scratch, and this curiosity led me to build a very simple database from sctach and I ended up doing full end-to-end database development from requirements to deriving isnights with queries. This was my process : Requirements --> ERD Design --> Relational Model --> Crow Foots Notation --> Sample Data Tables --> SQL script (creating and loading data) --> checking and performing sql queries to derive insights.
